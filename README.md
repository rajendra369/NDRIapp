#ATMOSYN – Atmospheric SYNthesis

Author: Engineer, Rajendra Kr. Shrestha

This repository hosts the Atmosyn mobile application and its associated backend services for meteorological research.

## Project Structure

- **[Atmosyn/](./Atmosyn)**: The primary mobile application for meteorological data collection.
- **[backend/](./backend)**: Backend services and API for data synchronization.
- **Legacy Prototypes**: The original web-based prototype remains available in the `Atmosyn-Web-Prototyp` branch.

## Getting Started

### Atmosyn (Mobile App)
To run the Atmosyn application:
1. Navigate to the `Atmosyn` directory.
2. Run `flutter pub get`.
3. Use `flutter run` to launch on your device/emulator.

### Backend
To run the backend:
1. Navigate to the `backend` directory.
2. Follow the instructions in the backend folder.
3. Or use the shortcut `run_backend.bat` in the root folder.
